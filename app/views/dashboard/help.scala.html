Introduction to Learn Lab


Events

You can create events including a group of students. An event has three stages. Each stage is timed. Every stage has a script which describes the question and how the students can participate.


Stage 1: This stage has a multiple choice question. The student individually answers this question.

Stage 2: In this stage the students belonging to the same event, chat using instant messaging feature and discuss what they think is the correct answer.

Stage 3: In this stage, the student will get a chance to correct their answers and justify the same

Stage 4: In this stage, the students will face a followup question



Statistics

You can view the statistics of an event or statistics of an individual student by following appropriate links on the left navigation bar. The statistics help to understand a student`s standing in the course and whether or not collaboration fostered learning in the event.



Live Chat Feed

This feature lets you choose an event and observe the live chat discussions students are having in Stage 2. This also lets the instructor to intervene and send messages to direct students in the right path, if necessary



Past Event Discussions

This feature lets you choose a past event and view the discussions which have happened in it. Using this feature,events can be better designed to foster learning through collaboration.



Event Summary

You can view the summary of the event by selecting an appropriate event. The summary includes details such of stages in which majority of students were successful, whether outcome of collaboration was positive or negative, the most upvoted user etc. These details provide more clarity to the instructor to understand if the event was successful.
